# Release Notes: Hawaii Travel Album v1.0 (Instagram Edition) 🌺📸

이번 업데이트는 사용자의 미디어 감상 경험을 극대화하기 위해 **"Instagram-like Feed"** 스타일로 UI/UX를 전면 개편했습니다. 
이제 모바일과 데스크탑 환경 모두에서 일관되고 현대적인 피드 경험을 제공합니다.

---

## 🚀 주요 변경 사항 (Key Changes)

### 1. 📱 완벽한 피드형 뷰 도입 (Universal Carousel)
- **Grid View 제거**: 기존의 바둑판식(Grid) 이미지 배열을 제거하고, 사진을 한 장씩 크게 감상할 수 있는 **Carousel View**를 전면 도입했습니다. (하단 갤러리 섹션 삭제됨)
- **반응형 디자인**: 
  - **Mobile**: 화면을 꽉 채우는 몰입감 있는 뷰 (Height: 350px).
  - **Desktop**: 시원시원한 고해상도 뷰 (Height: 500px).
- **Infinite Loop**: 마지막 사진에서 다음으로 넘기면 처음으로 돌아가는 무한 루프 네비게이션을 구현하여 끊김 없는 탐색이 가능합니다.

### 2. 🖱️ 직관적인 네비게이션 (Intuitive Navigation)
- **Smart Arrows**: 데스크탑 환경에서 마우스를 올리면 **이전(<) / 다음(>)** 화살표가 나타나 클릭 한 번으로 편하게 이동할 수 있습니다.
- **Pagination Dots**: 모바일 감성을 살린 페이지네이션 도트를 하단에 배치하여 현재 보고 있는 사진의 위치를 직관적으로 파악할 수 있습니다.
- **Touch & Swipe**: 터치패드나 모바일 터치 스크린에서의 **스와이프 제스처**를 완벽하게 지원합니다.

### 3. 🎥 심리스 미디어 인터랙션 (Seamless Media Interaction)
- **Inline Experience**: 번거로운 팝업(Lightbox)을 제거하고, 피드 내에서 바로 콘텐츠를 즐길 수 있도록 개선했습니다.
- **Auto-Play Video**: 동영상이 피드에 노출되면 자동으로 재생되어 생동감을 더합니다 (기본 음소거).
- **Click-to-Mute**: 동영상을 클릭하면 소리를 켜거나 끌 수 있습니다(Toggle). 직관적인 **🔇/🔊 아이콘**으로 상태를 표시합니다.

---

## 🛠️ 콘텐츠 업데이트 (Content Updates)
- **다이아몬드 헤드**: 다이아몬드 헤드 정상 등반의 성취감을 담은 설명이 업데이트되었습니다.
- **타임라인 정제**: 몰입을 방해하는 렌터카 관련 불필요한 스크린샷 및 중복 영상을 제거하고, 자동 생성된 문구를 자연스럽게 수정했습니다.
- **데이터 무결성**: 생성 스크립트(`generate-timeline.js`)를 개선하여 메타데이터가 없는 파일도 안정적으로 처리하도록 수정했습니다.
